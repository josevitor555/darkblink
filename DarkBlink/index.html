Front-End é a parte visual de um site moderno; Responsável por layout, cores, fontes e todos os elementos virtuais. O/A desenvolvedor(a) fica na parte de interatividade, na qual isso envolve HTML5, CSS3 e JavaScript. Além disso, o Front-End pode interagir com outras tecnologias do Back-End.
Back-End é a parte lógica "por trás" do site ou aplicativo, ou seja, a parte que faz toda lógica e processamento de dados, bem como a conexão com banco de dados, servidores e outros sistemas que o usuário "não vê". O/A desenvolvedor(a) trabalha com Python, PHP, Ruby e Java (as principais); Essa parte torna o site ou aplicativo mais escaláveis e confiáveis.
Existem diversos frameworks de Front-End & Back-End atualmente. Vajamos alguns dos principais framework do Front-End: ReactJs, Angular, Vue.js, Booststrap, Materialize, Foundation. E, por último, framework do Back-End são: Node.js, Laravel, Django, ruby on Rails, Express.js e Flask.

1991 - Ano de lançamento

HTML

Sua primeira versão foi datada em 1991, como HTML 1.0. Hyper Text Markup, não é uma linguagem de programação, e sim de marcação HyperText e HyperLink. Fundamental para a criação de sites e aplicações web.

1995 - Ano de lançamento

CSS

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo (en-US) usada para descrever a apresentação de um documento escrito em HTML ou em XML (incluindo várias linguagens em XML como SVG, MathML ou XHTML).

1996 - Ano de lançamento

JavaScript

Origem e História. Concebido como uma linguagem do lado do servidor por Brendan Eich (então empregado pela Netscape Corporation), o JavaScript logo chegou ao Netscape Navigator 2.0 em setembro de 1995.

1991 - Ano de lançamento

Java

Uma das suas principais características do Java é a portabilidade, ou seja, programas escritos através desta linguagem podem ser exeutados em diferentes plataformas - isso graças a uma máquina virtual do Java, a JVM (Java Virtual Machine).

1995 - Ano de lançamento

MySQL

MySQL é um RDBMS de código aberto ou serviço usado na criação e gerenciamento de bancos de dados baseados no modelo relacional.

1991 - Ano de lançamento

Python

Em 1991, o matemático Guido van Rossum criou a linguagem de programação Python, escreveu na documentação oficial da linguagem um breve histórico do início do seu desenvolvimento.
